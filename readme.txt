Steps To Deploy The Application

1. Clone the application
    git clone https://github.com/SLIIT-Y3-Projects/DS-Ayurvedic-Shopping-Platform

2. Go to the project directory
    cd DS-Ayurvedic-Shopping-Platform

3. Copy the .env files from "env-files" folder provided with the submission

4. Run the application
    Method 1 (Using Yarn)
        double click on the start.bat file
